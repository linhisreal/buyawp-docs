## ✅ System Requirements

{% hint style="warning" %}
Before proceeding with the installation, ensure your system meets these requirements:

* Windows 10/11 (64-bit)
* Administrative access
* space required: 15.7 MB (might changed in future)
* Stable internet connection
{% endhint %}
