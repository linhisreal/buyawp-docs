## 🔑 License Activation

{% tabs %}
{% tab title="Activation Steps" %}

### Post-Purchase Activation

1. Copy your license key
2. Join our [Discord Server](https://{{ supportServer }})
3. Go to commands channel
4. Use `/claim` command
5. Enter your license key
6. Follow bot instructions
7. Wait for confirmation

{% hint style="warning" %}
**Important:**

* Keep your license key safe
* Don't share your key
* One key per user

{% endhint %}
{% endtab %}

{% tab title="Activation Issues" %}

### Common Activation Problems

If activation fails:

1. Verify key format
2. Check if key is already used
3. Ensure Discord account is verified
4. Try again after 5 minutes
5. Contact support if issues persist

{% endtab %}
{% endtabs %}
